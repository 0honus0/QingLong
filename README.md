# hifiniSignIn
https://www.hifini.com 自动签到

基于 whyour/qinglong:debian docker image

```
HIFINI_COOKIES      Cookies 
HIFINI_USER         用户
HIFINI_PASS         密码

QL_URL              QL服务端 Url
QL_CLIENT_ID        QL服务端 ClientId
QL_CLIENT_SECRET    QL服务端 ClientSecrte

TG_BOT_TOKEN        TG_Bot_Token
TG_USER_ID          TG_User_Id
```
